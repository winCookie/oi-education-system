# 博客系统快速开始指南

## 系统已部署

博客系统已成功部署到Docker容器中，现在可以立即使用！

### 访问地址
- **前端**：http://localhost
- **后端API**：http://localhost:3000
- **博客入口**：http://localhost/blog

---

## 立即开始 3 步

### 步骤 1：创建博客分类（管理员）

1. 使用管理员账号登录
2. 访问：http://localhost/blog/manage
3. 点击"新建分类"按钮
4. 创建以下5个基础分类：

| 分类名称 | 描述 |
|---------|------|
| 学习笔记 | 记录学习过程中的思考和总结 |
| 解题思路 | 分享题目的解法和思路 |
| 经验分享 | 分享学习和备赛经验 |
| 竞赛动态 | 竞赛资讯、获奖喜报、赛后总结 |
| 技术教程 | 工具使用、环境配置、调试技巧 |

### 步骤 2：发布第一篇文章

#### 管理员发布官方文章

1. 点击导航栏"博客"
2. 点击"写文章"按钮
3. 填写标题：`欢迎来到OI博客`
4. 选择分类：`经验分享`
5. 添加标签：`官方` `公告`
6. 编写内容：
```markdown
# 欢迎来到OI博客

欢迎各位同学来到OI教学系统的博客平台！

这里是大家分享学习心得、解题思路和竞赛经验的地方。

## 发布规则

- 学生发布的文章需要经过管理员审核
- 教师可以直接发布文章（带教师标识）
- 管理员发布的文章会显示官方标识

## 内容限制

- 文章标题：最多100字符
- 文章内容：最多5000字符
- 文章标签：最多5个

祝大家学习愉快！
```

7. 点击"发布文章"

#### 教师测试

1. 使用教师账号登录（如 teacher01）
2. 创建一篇技术教程文章
3. 直接发布（会显示"教师"标识）

#### 学生测试

1. 使用学生账号登录（如 student01）
2. 创建一篇学习笔记
3. 提交审核
4. 等待管理员审核

### 步骤 3：测试互动功能

1. 浏览发布的文章
2. 点赞文章
3. 发表评论
4. 回复评论
5. 收藏文章

## 故障排查

### 问题1：博客入口找不到
**解决**：刷新页面，查看导航栏是否有"博客"菜单

### 问题2：无法创建文章
**检查**：
- 是否已登录
- 标题和内容是否为空
- 字符长度是否超限

### 问题3：学生文章无法发布
**说明**：这是正常的，学生文章需要提交审核，由管理员审核后才能发布

### 问题4：没有分类可选
**解决**：管理员先创建分类（访问 /blog/manage）

### 问题5：评论无法发表
**检查**：
- 是否已登录
- 评论内容是否为空
